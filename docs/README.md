## ❗예외 상황

- 다리 길이 입력
  - 숫자가 아닌 값 입력
  - 3 - 20사이 범위가 아닌 값 입력
- 이동할 칸 입력
  - U, D가 아닌 값 입력
- 재시작 여부
  - R, Q가 아닌 값 입력
