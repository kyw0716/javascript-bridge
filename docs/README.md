## ❗ 예외 사항 (오류 발생 경우)

- 다리 길이 입력
  - 숫자가 아닌 값이 포함되어 있을 경우
  - 3 - 20의 범위를 벗어난 경우
- 이동할 칸 입력
  - U, D 이외의 값이 포함되어 있을 경우
- 재시작 여부 입력
  - R, Q 이외의 값이 포함되어 있을 경우
